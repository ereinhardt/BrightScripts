# BrightScripts

Various BrightScript `autorun.brs` applications for BrightSign media players.

by Erik Anton Reinhardt.<br>
[MIT License]

---

## Contents:

- `BrightSign-AV-Sync`: A set of BrightSign scripts (Master and Client) that synchronize audio and video playback across BrightSign players by sending messages over Ethernet.
- `Brightsign-Solo-Loop`: A simple BrightSign script that loops a single video or audio file on one player.
- `BrightSign-UDP-Trigger-Listener`: A BrightSign script that listens for specified UDP messages to start video and audio playback.
- `BrightSign-Website`: A BrightSign script that displays a specified website.

**Note**: <br>

- Detailed documentation and specific application descriptions can be found in each script’s directory, as well as in the comments within the script.
- Warning: Apple’s TextEdit app will break your scripts when you edit them! It will silently convert straight quotes to “smart quotes” and other substitutions that corrupt the code.
- The SD card needs to be formatted as FAT32 or exFAT.
