# BrightSign-Solo-Loop (v.1.0-2-2026)

A simple BrightSign script that loops a single video or audio file on one player – no network or sync required.

by Erik Anton Reinhardt (Based on Zach Poff).<br>
[MIT License]

---

**Usage**: <br>

- Copy `autorun.brs` to the root of the BrightSign microSD card.
- Place a single video or audio file (MOV, MP4, MPG, VOB, MP3, WAV, OGG, TS) in the same location; it will be detected automatically.
- Configure the output settings (resolution, scale mode, volume) as described in the script.
